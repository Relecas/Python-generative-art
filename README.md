# 🎨 Python Generative Art

Generative art created with Python and matplotlib. Mathematical patterns and visual design pieces built entirely with code.

---

## Notebooks

### [`generative_art.ipynb`](./generative_art.ipynb) — Mathematical Patterns
| Piece | Description |
|---|---|
| Mandelbrot Set | Ultra-zoom into seahorse valley with smooth fire-ocean coloring |
| Spirograph | Four hypotrochoid curves with rainbow phase gradients |
| Fractal Tree | Depth-11 recursive branching, bark-to-gold color shift |
| Lissajous Figures | 12-panel grid of phase-swept harmonic curves |
| Voronoi Mosaic | Neon stained-glass tessellation |

### [`design_art.ipynb`](./design_art.ipynb) — Visual & Artistic
| Piece | Description |
|---|---|
| Watercolor Wash | Layered translucent blobs that bleed like wet paint on paper |
| City Skyline | Procedural night skyline with randomized lit windows and water reflection |
| Aurora Borealis | Gaussian-blurred sine waves simulating northern lights |
| Botanical Illustration | Ink-style flowers and leaves with hand-drawn veins |
| Retro Sunset | 80s synthwave grid, banded sun, palm silhouettes, scanlines |
